# AskMyDoc

AskMyDoc is a user-friendly document analyst powered by LLMs and RAG (Retrieval-Augmented Generation), designed to simplify your understanding of complex documents. Upload PDFs and DOCX files, ask questions in natural language, and get instant answers.

## Features
-	AI-Powered Q&A: Ask questions about your documents — get intelligent, contextual answers.
-	Multi-File Upload: Upload multiple PDFs or DOCX files into the same chat session.
-	Multi-Chat Support: Working on different projects? Start a new chat with a different user_id to separate your sessions.
-	Clear Chat History: Done with a project? Use the simple “clear” command to erase document history.
